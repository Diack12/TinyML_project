# Lien vers le Projet Edge Impulse

## Accès au projet

Voici le lien vers le projet Edge Impulse pour la classification de composants :

**https://studio.edgeimpulse.com/public/879994/live**

### Informations du projet

- **Nom** : Composants Classification 
- **Modèle** : Classification d'images 64×64 en grayscale
- **Classes** : Background, Led, Resistor, PushButton
- **Compilateur** : EON Compiler (optimisé pour embarqué)
- **Cible** : Arduino Nano 33 BLE Sense

### Utilisation

Ce lien permet de :
- ✅ Consulter les données d'entraînement
- ✅ Voir les métriques de performance
- ✅ Tester le modèle en direct
- ✅ Télécharger la librairie Arduino mise à jour (si nécessaire)

### Important

Pour télécharger la librairie Arduino depuis ce projet :
1. Accéder au projet via le lien ci-dessus
2. Aller à **Deployment** → **Arduino Library**
3. Sélectionner **Arduino Nano 33 BLE Sense**
4. Cocher **Enable EON Compiler**
5. Cliquer **Build** et télécharger le fichier `.zip`

> Le fichier ZIP est également fourni dans le dossier `arduino_camera_classification/` du projet.
